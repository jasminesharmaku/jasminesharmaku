# Kushma Art Project Website

A professional, highly interactive website for Kushma Art Project (KAP) featuring premium UI/UX design with modern web technologies.

## 🎨 Features

### Design & Visual Elements
- **Glassmorphism Effects**: Beautiful glass-morphic cards with backdrop blur effects
- **Parallax Scrolling**: Smooth parallax effects on hero sections and backgrounds
- **Custom Cursor**: Interactive custom cursor with follower effect (desktop only)
- **Smooth Animations**: Fade-in, slide-up, and transition animations throughout
- **Loading Screen**: Elegant animated loader with KAP branding
- **Responsive Design**: Fully responsive across all devices (mobile, tablet, desktop)

### Pages & Navigation
1. **Home Page**
   - Animated hero section with video background
   - Featured initiatives with glassmorphic cards
   - Mission statement with animated statistics
   - Scroll indicator animation

2. **Shatabdhi Ghar Page**
   - Page hero with parallax background
   - Detailed information about the historic house
   - Image gallery preview
   - Visit information sidebar

3. **Programs & Residencies Page**
   - Program cards with icons and descriptions
   - Artist residency information
   - Youth workshops and cultural exchange programs
   - Application section

4. **Impact & Research Page**
   - Impact metrics with animated counters
   - Research documentation section
   - Community testimonials
   - Statistical visualizations

5. **Donate Page**
   - **Urgent Goal Highlighted**: "Fund the Kitchen" campaign
   - Progress bar animation
   - Donation amount selection
   - Multiple giving options
   - Impact breakdown by donation level

6. **Contact Page**
   - Contact form with validation
   - Contact information cards
   - Animated map marker
   - Social media links

### Interactive Features
- **Single Page Application**: Smooth page transitions without reload
- **Mobile Menu**: Hamburger menu for mobile navigation
- **Scroll Animations**: Elements animate into view on scroll
- **Hover Effects**: Interactive card tilts and button effects
- **Form Validation**: Built-in contact form
- **Scroll-to-Top Button**: Appears after scrolling down
- **Statistics Counter**: Animated number counting
- **Gallery Lightbox**: Click images to view in lightbox
- **Micro-interactions**: Button ripple effects and transitions

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with custom properties, animations, and responsive design
- **JavaScript (ES6+)**: Vanilla JavaScript for all interactions
- **Google Fonts**: Montserrat and Playfair Display typography
- **Intersection Observer API**: For scroll-based animations
- **CSS Grid & Flexbox**: Modern layout techniques

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 968px
- Desktop: > 968px

## 🎯 Key Design Principles

1. **User Experience**: Intuitive navigation with clear call-to-actions
2. **Performance**: Optimized animations and lazy loading support
3. **Accessibility**: Keyboard navigation support and semantic HTML
4. **Visual Hierarchy**: Clear content structure with typography scale
5. **Color Psychology**: Carefully chosen color palette reflecting art and community

## 🚀 Getting Started

### Local Development

1. Clone the repository
2. Open `index.html` in a web browser
3. Or use a local server:
   ```bash
   python3 -m http.server 8000
   # Visit http://localhost:8000
   ```

### Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
